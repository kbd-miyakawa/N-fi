●動作環境
　Windows 7 〜 Windows 11 で多分大丈夫です。

●インストール
　適当なディレクトリに以下のファイルを置いてください。

　NyanFi.exe 64ビット VCLスタイル版 NyanFi 本体

　NyanFi.chm ヘルプファイル
	 内容が表示されない場合は、ファイルのプロパティでブロックを解除してください。

　shexe.exe
    パラメータで指定した実行ファイル名を起動するだけの単純なプログラムです。
    VCLスタイルを使用している場合、ネットワーク先の実行ファイルを開く際にセキュリティの警告ダイアログの実行ボタンが機能しない現象に対処するためのものです。

　アーカイブを扱うには、別途 7-zip64.dll、tar64.dll、unrar64j.dll/unrar64.dll が必用です。

　C/Migemo に対応しています。利用する場合は、migemo.dll (64bit版)や辞書ファイルをインストールしてください。

　xd2txlib.dll (xdoc2txt のDLL版)に対応しています。各種バイナリ文書に対して、テキストビュアー表示、文字列検索(GREP)などを行えます。

　テキストビュアーの HelpCurWord コマンドで、カーソル位置の単語を指定したヘルプで検索するには、hhh.exe (saitou155 氏作)が必要です。
　あらかじめ、パスの通ったディレクトリ(hh.exe がある C:\Windows など)にインストールしておいてください。 

　なお、ファイルメニュー - [更新の確認]で、最新版のチェックおよび更新が可能です。

●アンインストール
　インストールしたファイルおよび INIファイルなどを削除して下さい。
　レジストリは使用していません。
